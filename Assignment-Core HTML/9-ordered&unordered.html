1. p
    • Use this tag to create paragraphs of text.
    • Always start a new paragraph with this tag, and avoid using multiple p tags for the same paragraph.
    • Add line breaks or spacing between paragraphs using CSS styles, rather than multiple p tags.
  2. b
    • Use these tags to bold text for emphasis or importance.
    • Use sparingly to avoid overwhelming the reader with too much emphasis.
  3. s
    • Use these tags to strike through text to indicate that it is no longer relevant or accurate.
    • Use sparingly to avoid confusing the reader with too much strikethrough text.
