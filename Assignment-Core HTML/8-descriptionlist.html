HTML
Markup language used for creating the structure and content of web pages.
CSS
Style sheet language used for designing and styling the layout and appearance of web pages.
JavaScript
Programming language used for creating interactive and dynamic effects on web pages, such as animations and form validations.
React
JavaScript library used for building user interfaces, including complex and interactive components.
Node.js
JavaScript runtime used for developing server-side applications, allowing for JavaScript to be used both on the front-end and back-end.
Express
Web application framework for Node.js used for creating server-side APIs and handling HTTP requests and responses.
MongoDB
NoSQL document database used for storing and retrieving data in a flexible and scalable way, particularly for web applications that handle large amounts of unstructured data.
MySQL
Relational database management system used for storing and managing structured data, particularly for web applications that require complex data relationships and transactions.
Git
Version control system used for managing and tracking changes to code and files, particularly for collaborative software development.
